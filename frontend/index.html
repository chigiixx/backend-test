Обо мне

Сейчас работаю веб-разработчиком в «ИТР-Строй» (Нижний Новгород) — с нуля спроектировала и запустила в продакшен адаптивный сайт-SPA на React 19, TypeScript и Tailwind CSS, с интерактивным калькулятором стоимости услуг и формой обратной связи с уведомлениями. Спроектировала структуру базы данных PostgreSQL, настроила резервное копирование и безопасность, довела Lighthouse до 91–94.5 и полное соответствие WCAG 2.1 AA.

До этого успела поработать с моделированием бизнес-процессов (IDEF0, BPMN) на стажировке по проектированию информационных систем, администрированием серверов и баз данных в роли младшего системного администратора и разработкой desktop-приложений на C#. Сейчас самостоятельно изучаю Python и расширяю backend-стек — этот сайт и API под ним как раз пример того, что получилось.
